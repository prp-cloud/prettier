#### Keep curly braces and `from` keyword in empty `import` statements (#15756 by @fisker)

<!-- prettier-ignore -->
```js
// Input
import { } from 'foo';
import { /* comment */ } from 'bar';

// Prettier stable
import {} from "foo";
import /* comment */ "bar";

// Prettier main
import {} from "foo";
import {} from /* comment */ "bar";
```
